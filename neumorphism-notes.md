# Neumorphism / Soft UI Design Parameters

## Color Palette
- Background: #e8e8e8 (light gray, the base)
- Surface: #f0f0f0 / #fafafa
- Dark shadow: #d1d1d1 / #d4d4d4
- Light shadow: #ffffff
- Accent blue: #c8e0f4 (soft blue)
- Accent pink: #f5e0e8 (soft pink)

## Shadow Formula (Soft UI)
- Raised element: box-shadow: -5px -5px 15px #ffffff, 5px 5px 15px #d1d1d1
- Pressed/inset: box-shadow: inset -3px -3px 8px #ffffff, inset 3px 3px 8px #d1d1d1
- Transition: 150ms smooth press

## Shape
- Border radius: 12-16px (rounded)
- Monochromatic (single color family)

## Typography
- Sans-serif (Inter or system)
- Subtle, muted text colors

## Usage
- Health/wellness apps, meditation platforms, fitness trackers
- Minimal interactive UI

## Tomato Clock Adaptation (for lifestyle blog)
- Use warm beige/cream base: #ede8e3 (matches blog's warm white)
- Raised card: box-shadow: -6px -6px 16px #ffffff, 6px 6px 16px #ccc5bc
- Pressed button: box-shadow: inset -3px -3px 8px #ffffff, inset 3px 3px 8px #ccc5bc
- Accent: warm rose #e8a0a0 (matches lifestyle category color)
- Timer ring: SVG circle with stroke animation
- Size: ~220px wide widget (phone-sized reference from screenshot)
