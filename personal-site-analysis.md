# 个人网站设计分析 — thebestisyettocome.wang

## 背景色
- 主背景：暖米白 / 奶油亚麻色，类似 oklch(0.97 0.012 75) — 非纯白，有温度
- 大理石纹理感背景（浅米色 + 细腻纹理）

## 配色方案
- 主色调：暖米白背景 #f5f0e8 / #ede8de
- 强调色（italic 标题）：天蓝/钢蓝色 — 类似 #4a90c4 或 oklch(0.62 0.12 240)
- 文字主色：深炭黑 #1a1a1a / #2a2520
- 次要文字：中灰 #6b6560
- 边框/分割线：浅米灰 #d8d0c8
- CTA 按钮：深色填充（深炭/黑）+ 白字；空心按钮（透明+深色边框）
- 标签/Tag：浅米色背景 + 小字

## 字体
- Logo/品牌名：粗衬线体（类 Playfair Display Bold / DM Serif Display）
- 导航：无衬线细体，字间距宽（Montserrat / DM Sans，letter-spacing: 0.1em）
- 大标题 H1：超大衬线体，混合正体（黑色）+ 斜体（蓝色强调）
- 正文：无衬线，轻量，行距宽松（Inter / DM Sans，font-weight: 400）
- 引用块：斜体衬线体
- 章节编号：细小无衬线，字间距极宽

## 排版风格
- 不对称布局：左侧大标题 + 右侧内容卡片
- 大量留白，内容密度低
- 章节编号（01 / 02 / 03）右对齐，营造杂志感
- 细横线分割，极简
- 卡片：白色/浅米色背景，无边框，轻微阴影

## 设计语言关键词
- 奢侈品牌感（Breitling / Estée Lauder 气质）
- 极简主义，大量留白
- 衬线 + 无衬线混搭
- 暖色调米白底色
- 蓝色斜体作为点睛强调色
- 硬朗但优雅（直线、精准间距）

## 博客网站融合策略
- 背景色：从纯白 #ffffff 改为暖米白 #f7f3ee（更有质感）
- 强调色：引入钢蓝 #4a8fc0 作为分类标签、链接 hover 色
- 字体：保留 Playfair Display（标题）+ Montserrat（导航），正文改为 DM Sans（更轻盈）
- 导航：增加字间距，更精致
- 卡片：轻微暖色阴影，去掉冷灰色
- 保持 Soft UI 温柔感：圆角、柔和阴影、渐变保留
