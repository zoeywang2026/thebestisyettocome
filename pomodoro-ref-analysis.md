# 番茄钟参考设计分析

## 整体风格
- 极简主义，大量留白
- 背景：极浅粉米白色（接近 #f5f0ee）
- 整体氛围：冥想/专注空间感（"FOCUS SPACE"）

## 核心视觉元素
- **主体形状**：竖向椭圆形卡片（蛋形），白色/浅色，带轻微阴影
- 椭圆顶部有细小装饰图标（类似沙漏/铃铛）
- 椭圆内部：
  - 顶部：斜体衬线字 "Focus"（模式标签）
  - 中央：超大无衬线数字 "25:00"（粗体，深色）
  - 底部：三个控制按钮（Reset / START / Skip）
    - START 按钮：细边框圆角矩形，字间距宽
    - Reset 和 Skip：图标按钮，无背景

## 底部统计区
- 三列数据：TODAY (03) / TOTAL (75m) / STREAK (#1)
- 细小无衬线字体，字间距宽，颜色浅灰

## 底部导航栏
- 4个图标：闹钟/计时器、音乐/节拍、心形、设置
- 极细线条图标，居中排列

## 配色
- 背景：#f5f0ee（极浅粉米白）
- 椭圆卡片：白色 #ffffff 或 #faf8f7，轻微阴影
- 主文字：深炭色 #2a2520
- 次要文字：中灰 #9a9090
- 按钮边框：浅灰 #d8d0cc
- 无强调色，极度克制

## 字体
- 模式标签：斜体衬线体（Playfair Display italic）
- 时间数字：大号无衬线粗体（类 DM Sans Bold 或 Montserrat Bold）
- 按钮文字：极细无衬线，字间距宽（Montserrat Light）
- 统计标签：极小无衬线，全大写，字间距宽

## 设计关键词
- 蛋形/椭圆主体
- 极简留白
- 冥想/禅意
- 无强调色
- 衬线斜体 + 无衬线数字混搭
- 与个人网站风格高度契合
